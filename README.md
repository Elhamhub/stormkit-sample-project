## Sample Project

This project is an example project to be deployed on [Stormkit](https://www.stormkit.io). 

## Features

✅ api
✅ redirects (see [redirects.json](./redirects.json))

## Testing locally

Starts the frontend server: 

```bash
npm run start
```

Starts the API server:

```bash
npm run dev:api
```
